## 思考题

{% ex %}

给定 $A \in \mathbb C^{n \times n}$. 证明存在唯一的幂零矩阵 $N$ 与可对角化矩阵 $D$ 使得

$$\begin{equation}
ND = DN,\quad A = D + N.
\end{equation}$$

{% pf %}
$N$ 与 $D$ 存在性由 Jordan 标准型得到. 下证明唯一性. 设 $A = N' + D'$ 是另一种分解, 则容易发现存在多项式 $f$ 与 $g$ 使得

$$\begin{equation}
f(A) = N, \quad g(A) = D.
\end{equation}$$

因此 $N - N' = D-D'$ 是可对角化的幂零矩阵, 这只能是零矩阵. 由此得到唯一性.

{% endpf %}

{% endex %}

{% ex %}
若数域上的矩阵 $A$ 与 $X$ 满足 $AX-XA = X$, 证明 $X$ 幂零.

{% pf %}
直接地, $tr(X) = tr(AX)-tr(XA) = 0$. 可以归纳地证明 $tr(X^k) = 0$ 对任意正整数 $k$ 成立. 由 Vitae 定理知 $X$ 的特征值多项式为 $x^n$. 因此 $X$ 幂零.
{% endpf %}

{% endex %}
