## 特征根估计: 圆盘定理

### 圆盘定理的证明

圆盘定理用于估计矩阵的特征根. 以下所有矩阵是 $\mathbb C$ 上方阵.

{% thm %}
设 $A=(a_{ij})$ 是 $n\times n$ 矩阵. 对每个 $i$, 总有

$$\begin{equation}
|a_{ii}| > \sum_{j\neq i} |a_{ij}|.
\end{equation}$$

则 $A$ 可逆.

{% pf %}
见 [HW7](HW7-solu#行满秩矩阵的一个例子).
{% endpf %}

{% endthm %}

{% def %}
(复矩阵的 Gershgorin 圆盘). 给定 $A\in \mathbb C^{n\times n}$. 对 $1\leq k\leq n$, 定义 $A$ 的第 $i$ 个列圆盘为

$$
D_i= \left\{z: |z-a_{i,i}|\leq \sum_{1\leq j\leq n,\text{ 且 }j\neq i} |a_{i,j}| \right\}.
$$

以上,

* 圆盘的中心是 $a_{i,i}\in \mathbb C$,
* 圆盘的半径是 $\sum_{1\leq j\leq n,\text{ 且 }j\neq i} |a_{i,j}|$​.

{% enddef %}

{% ex %}
矩阵 $\begin{pmatrix}5+i& 3-2i& i\\ 2-i & 7i& -2+i \\ 2i-1 & i & -1-i\end{pmatrix}$, 特征根与 Gershgorin 圆盘分布如下图所示:

<center><img src="https://zhangchenchengsjtu.github.io/MATH1205H-04-HW/assets/svg/G_Disc_Sample.svg"></center>

{% endex %}

{% pro %}
给定复方阵的特征对 $(\lambda, v)$. 假定 $v$ 中第 $i$ 个分量模长最大, 证明 $\lambda \in D_i$​.

{% pf %}
设 $v=(v_1, v_2, \ldots, v_n)^T$. 考虑 $\lambda v = Av$ 的第 $i$ 行:

$$\begin{equation}
\lambda v_i = \sum_{j=1}^n a_{i,j} v_j = a_{i,i} v_i + \sum_{j\neq i} a_{i,j} v_j.
\end{equation}$$

从而

$$\begin{equation}
|\lambda - a_{ii}| = \left|\sum_{j\neq i} a_{i,j} \frac {v_j}{v_i}\right| \leq \sum_{j\neq i} |a_{i,j}| \left|\frac {v_j}{v_i}\right| \leq \sum_{j\neq i} |a_{i,j}|.
\end{equation}$$

{% endpf %}

{% endpro %}

{% cor %}
$\bigcup_{i=1}^nD_i$ 中包含了 $A$​​ 的所有特征值.
{% endcor %}

{% ex %}
对角矩阵的所有圆盘半径为 $0$. 圆盘定理显然成立.
{% endex %}

{% pro %}
请计算

$$\begin{equation}
\begin{pmatrix}
    4i+12&6i-2&-2i-6\\
    6i+8&-2i+10&5i-3\\
    -14i-2&12i-12&8i+18
\end{pmatrix}
\end{equation}$$

的特征根, 并画出相应的 Gershgorin 圆盘. 以此验证 Gershgorin 圆盘定理.

{% pf %}
$10-10i$, $10+10i$ 与 $20+10i$ 是特征根.

<center><img src="https://zhangchenchengsjtu.github.io/MATH1205H-04-HW/assets/svg/G_Disc_Ans.svg"></center>

{% endpf %}

{% endpro %}

{% thm %}
若 $A \in \mathbb C^{n \times n}$ 的 $n$ 个圆盘互不相交, 则每个圆盘中有且仅有一个特征根. 从而 $A$ 有 $n$ 个两两不同的特征根. 作为推论, $A$ 可对角化.

{% pf %}
记 $D$ 是 $A$ 的对角线. 考虑连续函数

$$\begin{equation}
F: [0,1] \to \mathbb C^{n\times n},\quad t\mapsto D + t(A-D).
\end{equation}$$

特别地, $F(0) = D$, $F(1) = A$. 直接地, $F(t)$ 的 $n$ 个圆盘不交.

- 一个分析学常识 (自行用 $\varepsilon$-$\delta$ 语言证明): 记 $\det (xI - F(t))$ 是一个关于 $x$ 的多项式, 其系数是参数 $t$ 的连续函数. 该多项式在复数域上的 $n$ 个根表示为 $n$ 个关于 $t$ 的函数, 记作 $\{x_i(t)\}_{i=1}^n$. 则每个 $x_i(t)$ 关于 $t$ 连续.

$D = F(0)$ 的所有圆盘中各有且仅有一个特征根 (即对角线元). 由连续性, 每个圆盘中始终有且仅有一个特征根. 由于圆盘互不相交, 故 $A=F(1)$ 有 $n$ 个两两不同的特征根. 因此 $A$ 可对角化.

{% endpf %}

{% endthm %}

### 与 Hamilton Cayley 定理的结合

{% thm %}
(Hamilton-Cayley 定理). 设 $A\in \mathbb C^{n\times n}$, 其特征多项式为 $p_A(x)=\det (xI - A)$. 则 $p_A(A)=0$​.

{% pf %}
对任意多项式 $f$, 总有 $f(PAP^{-1}) = Pf(A)P^{-1}$. 只需考虑 $A$ 的 Jordan 标准型. 这是显然的.
{% endpf %}

注: 一般域上的 Hamilton-Cayley 定理有以下几种证明方式. 最自然的想法是引入有限域扩张, 使得 $\det(xI - A)$ 能分解成一次因式的乘积, 然后对扩域上的 Jordan 标准型使用类似的证明. 另一种思路是使用伴随矩阵 (adjoint matrix)​ 的概念, 许多教材使用这种证明, 但动机不明. 这一定理的本质是 $\mathbb F^n$ 上的 $(n+1)$-次反对称形式消失, 见[去年习题课](https://www.mathcha.io/editor/9zzq8sn3tL2SVqI63wm4nIp3W50rsPxpdY0tpdL8Z).

{% endthm %}

{% prop %}
若 $AX = XB$, 且 $p_A(x) := \det(xI-A)$ 与 $p_B(x) := \det (xI - B)$ 互素, 则 $X = O$.

- 称 $p_A$ 与 $p_B$ 互素, 当且仅当存在多项式 $g$ 与 $h$ 使得 $g(x)p_A(x) + h(x)p_B(x) = 1$​.

{% pf %}
由特征多项式互素, $A$ 与 $B$ 至少一者可逆, 不妨假定为 $A$. 由归纳, 对任意多项式 $f$, 总有 $f(A)X = X f(B)$. 将 $f$ 取作 $B$ 的特征多项式, 得 $p_B(A)X = O$. 由互素, 存在 $g$ 与 $h$ 使得

$$\begin{equation}
X = g(A)p_A(A)X + h(A)p_B(A)X = O.
\end{equation}$$

{% endpf %}

{% endprop %}

{% cor %}
若 $A$ 与 $B$ 的特征多项式互素, 则对任意 $M$, $\binom{A \ M}{O \ B}$ 与 $\binom{A \ O}{O \ B}$ 总是相似的.

{% pf %}
由特征多项式互素, 线性映射

$$\begin{equation}
\mathbb C^{m\times n} \to \mathbb C^{m\times n},\quad X\mapsto AX - XB
\end{equation}$$

是单射, 从而是满射. 假定 $M$ 的原像是 $P$, 则

$$\begin{equation}
\binom{A \ O}{O \ B}\cdot \binom{I \ P}{O \ I} = \binom{I \ P}{O \ I} \cdot \binom{A \ M}{O \ B}.
\end{equation}$$

{% endpf %}

{% endcor %}

{% ex %}
考虑以下主对角占优的复矩阵构成的集合

$$\begin{equation}
\mathscr S := \{A \in \mathbb C^{n \times n} \mid a_{ii} > \sum_{j\neq i} |a_{ij}|,\ \forall 1\leq i\leq n\}.
\end{equation}$$

依照圆盘定理, $A \in \mathscr S$ 的所有特征根都有正的实部.
\\
对任意 $A,B \in \mathscr S$, 若 $A^2 = B^2$, 则 $A=B$.

- 证明: $A$ 与 $B$ 的特征根不同, 其特征多项式必互素. 由上文引理, $AX=X(-B)$ 只有零解. 由题设, $A(A-B)=(A-B)(-B)$. 从而 $A-B = O$.

{% endex %}

仿照上述证明, 试解决下列问题.

{% pro %}
定义集合

$$\begin{equation}
\mathscr T:= \{A \in \mathbb C^{n \times n} \mid \forall 1\leq i\leq n, \ a_{ii} \in \mathbb R,\ \text{且} \ |a_{ii}| >2\sum_{j\neq i} |a_{ij}|\}.
\end{equation}$$

对任意 $A,B\in\mathscr S$, 若 $A^3=B^3$, 则 $A=B$.

- 若嫌麻烦, 可以简单地取 $\mathscr T:= \{A \in \mathbb R^{n \times n} \mid  |a_{ii}| > 2\sum_{j\neq i} |a_{ij}|\}$. 证明过程没区别.

{% pf %}
$A,B \in \mathscr T$ 中矩阵的特征根落在 $(-\pi/6,\pi/6) \cup (5\pi/6,7\pi/6)$ 这个区段 (极坐标下). 记 $\omega = \frac{-1+\sqrt 3 i}{2}$, 则 $A$ 与 $\omega B$ 没有相同的特征根, $A$ 与 $\omega^2 B$ 也没有相同的特征根. 考虑等式

- $A(A^2+\omega AB+\omega^2 B^2) = (A^2+\omega AB+\omega^2 B^2)\omega B$,
- $A(A^2+\omega^2 AB+\omega B^2) = (A^2+\omega^2 AB+\omega B^2)\omega^2 B$.

从而

- $(A^2+\omega AB+\omega^2 B^2) = O$,
- $(A^2+\omega^2 AB+\omega B^2) = O$.

相减得 $AB=B^2$. 由矩阵可逆, 得 $A=B$.

{% endpf %}

{% endpro %}

{% thm %}
记 $\mathscr P$ 是特征值全为正实数的复矩阵构成的集合. 则对任意 $n$,

$$\begin{equation}
\mathscr P \to \mathscr P, \quad X \mapsto X^n
\end{equation}$$

是双射. 换言之, 每个 $\mathscr P$-中的矩阵有唯一的 $\mathscr P$ 中的 $n$ 次方根.

{% pf %}
记上述命题为 $\mathfrak P(n)$. 考虑以下归纳法:

1. $\mathfrak P(2)$ 正确,
2. 若 $\mathfrak P(a)$ 与 $\mathfrak P(b)$ 正确, 则 $\mathfrak P(a\cdot b)$ 正确,
3. 对素数 $p$, 若 $\mathfrak P(p-1)$ 正确, 则 $\mathfrak P(p)$ 正确.

显然 1. 与 2. 成立. 下证明 3. 即可. 记 $w$ 是 $p$-次单位根, 则对一切 $1\leq k\leq p-1$,

$$
\begin{align*}
&A(A^{p-1}+w^k A^{p-2}B +w^{2k}A^{p-3}B^2+\cdots + w^{(p-1)k}B^{p-1})\\
= \ &(A^{p-1}+w^k A^{p-2}B +w^{2k}A^{p-3}B^2+\cdots + w^{(p-1)k}B^{p-1})(w^k B).
\end{align*}
$$

因此, $A^{p-1}+w^k A^{p-2}B +w^{2k}A^{p-3}B^2+\cdots + w^{(p-1)k}B^{p-1}=O$. 写作线性方程组, 得

$$
(w^{i\cdot (j-1)})_{1\leq i\leq (p-1), \ 1\leq j\leq p}\cdot \begin{pmatrix}A^{p-1}\\ A^{p-1}B\\\vdots \\ B^{p-1}\end{pmatrix}_{p}
=\begin{pmatrix}O\\ O\\\vdots \\ O\end{pmatrix}_{p-1}.  
$$

此处, $\{A^{l}B^{p-1-l}\}_{0\leq l\leq p-1}$ 是 $p$ 个未知量. 左侧矩阵 $W_{(p-1)\times p}=(w^{i\cdot (j-1)})$ 形如增广矩阵 $(\mathbf 1 \ \ V)$, $V$ 是 $p$-阶 Vandermonde 方阵 (可逆). 因此, 存在行初等变换, 使得

$$
W=(\mathbf 1  \ \ V)\xrightarrow{\text{行初等变换}} (\mathbf v  \ \ I)\quad (\mathbf v \ \text{唯一, 由反方向的 $ref$ 算法}).
$$

由于行变换不改变列线性关系, 结合 $\mathbf 1\in N(W)$ 知 $\mathbf v= - \mathbf 1$. 此时, 方阵组的最后一个等式是 $A^{p-1}-B^{p-1}=O$. 这证明归纳步骤

- 对素数 $p$, 若 $\mathfrak P(p-1)$ 正确, 则 $\mathfrak P(p)$ 正确.

{% endpf %}

{% endthm %}
