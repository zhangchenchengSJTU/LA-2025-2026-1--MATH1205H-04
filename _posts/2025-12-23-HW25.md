## 线性空间巩固

### 例子

以下是一些线性空间的例子.

{% def %}
(形式幂级数). 设 $\mathbb F$ 是一个域. 形式幂级数是形如

$$\begin{equation}
f(x) = \sum_{n=0}^\infty a_n x^n, \quad a_n \in \mathbb F
\end{equation}$$


的表达式.

{% enddef %}

{% ex %}
(非常重要). 一般来说, 函数和映射是相同的概念. 但是:

- 多项式未必是函数. 例如, $x^2-x$ 与 $0$ 是二元域上相同的函数, 但它们作为多项式并不相同. (对数域而言, 多项式就是函数).
- 形式幂级数未必是函数. 形式幂级数只是一个形式上的记号, 不要求收敛性.

{% endex %}

形式幂级数的另一看法是数列.

{% ex %}
(数列空间). 以下三个线性空间彼此线性同构 (存在线性双射).

1. $\mathbb F$ 中全体的数列.
2. $\mathbb F$ 上的形式幂级数.
3. $\mathbb N$ 到 $\mathbb F$ 的全体映射.

($1$ 与 $3$ 等价). 实际上, 数列 $\{a_n\}_{n \geq 0}$ 对应函数

$$\begin{equation}
a_\bullet : \mathbb N \to \mathbb F, \quad n \mapsto a_n.
\end{equation}$$

($2$ 与 $3$ 等价), 实际上, 形式幂级数 $f(x)$ 对应函数

$$\begin{equation}
\mathbb N \to \mathbb F, \quad f \mapsto  \text{$f$ 中 $x^n$ 的系数}.
\end{equation}$$

{% endex %}

{% def %}
(多项式空间). 多项式空间 $\mathbb F[x]$ 是形式幂级数空间 $\mathbb F[[x]]$ 的子空间, 其包含仅有限项系数非零的形式幂级数. 多项式空间也线性同构与于数列空间的子空间, 包含仅有限项非零的数列.
{% enddef %}

{% ex %}
难以找到 $W \subseteq \mathbb F[[x]]$, 满足 $\mathbb F[x] \oplus W = \mathbb F[[x]]$.
{% endex %}

特别地, $\mathbb F[[x]]$ 与 $\mathbb F[x]$ 都是无限维 $\mathbb F$-向量空间.

{% ex %}
对有限维线性空间 $V$ 与线性映射 $\varphi, \psi : V \to V$, 映射 $\varphi \psi - \psi \varphi$ 不可能是恒等映射. 因为 $\mathrm{tr}(AB-BA) = 0$. 对无限维空间则不然, 例如, 取 $V = \mathbb R[x]$, 定义

- $\varphi : \mathbb R[x] \to \mathbb R[x],\quad f(x) \mapsto f'(x)$;
- $\psi : \mathbb R[x] \to \mathbb R[x],\quad f(x) \mapsto x f(x)$.

此时 $\varphi \psi - \psi \varphi$ 是恒等映射.

{% endex %}

{% pro %}
定义数列空间的左移与右移映射:

1. $L: \mathbb C[[x]] \to \mathbb C[[x]],\quad (x_0,x_1,\ldots ) \mapsto (x_1,x_2,\ldots)$;
2. $R: \mathbb C[[x]] \to \mathbb C[[x]],\quad (x_0,x_1,\ldots ) \mapsto (0,x_0,x_1,\ldots)$.

试解决以下问题 (主要是 4, 5 两题).

1. 试说明, $L$ 满而不单, $R$ 单而不满.
   - 这告诉我们, 对无限维线性空间的自同态, 单/满/同构未必等价!
2. 试说明, $L$ 与 $R$ 也是 $\mathbb C[x] \to \mathbb C[x]$ 的线性映射. 换言之, 子空间 $\mathbb C[x] \subseteq \mathbb C[[x]]$ 是 $L$ 与 $R$ 的不变子空间.
3. 记 $id$ 是恒等映射. 试说明, $L R = id$, 但 $R L \neq id$.
   - 这告诉我们, 若无限维线性空间的自同态有左逆, 则不一定有右逆 (反之亦然).
4. 求 $L$ 与 $R$ 的特征对, 以及特征空间的维数.
   - 这告诉我们, 无限维线性空间的线性映射未必有特征值, 即便是在复数域上.

{% endpro %}

{% pro %}
记 $V$ 是收敛实数列构成的空间, 则 $\mathbb R[x] \subseteq V \subseteq \mathbb R[[x]]$. 请找一个单而不满的线性映射 $\varphi : V \to V$ 与一个满而不单的线性映射 $\psi : V \to V$, 使得 $\varphi + \psi$ 是双射.
{% endpro %}

{% ex %}
(线性映射与方程). 假定 $V$ 是 $[0,1] \to \mathbb R$ 的连续函数构成的线性空间. 定义核函数

$$\begin{equation}
K(x,y) := \min(x,y) - xy.
\end{equation}$$

此时有线性映射

$$\begin{equation}
V \to V,\quad f(x) \mapsto \int_0^1 K(x,y) \cdot f(y)\operatorname{d}y.
\end{equation}$$

求这一线性映射的特征值与特征向量?

{% pf %}
若 $\lambda$​ 是特征值, 则化简得

$$\begin{equation}
\lambda f(x) = \int_0^x yf(y)\operatorname dy + \int_x^1 x f(y) \operatorname dy - \int_0^1 yx f(y) \operatorname dy.
\end{equation}$$

求导得

$$\begin{equation}
\lambda f'(x) = xf(x) + \int_x^1 f(y) \operatorname dy -xf(x)-\int_0^1 yf(y)\operatorname dy.
\end{equation}$$

再求导, 得 $\lambda f''(x) = -f(x)$. 此时得方程组 (显然 $\lambda =0$ 不是特征根) 

$$\begin{equation}
\lambda ''(x) = -f(x),\quad \lambda f(0) = \lambda f(1) = 0.
\end{equation}$$

若 $\lambda <0$, 则解得 $f(x) = a \cdot e^{x/\sqrt{-\lambda}} + b \cdot e^{-x/\sqrt{-\lambda}}$. 代入 $f(0)=f(1)=0$ 得 $a=b=0$.
\\
若 $\lambda > 0$, 则解得 $f(x) = a \cdot \sin \frac{x}{ \sqrt \lambda} + b \cdot \cos \frac{x}{ \sqrt \lambda}$. 代入 $f(0)=f(1)=0$ 得 $b=0$, 以及 $\sin \frac{1}{\sqrt \lambda} =0$. 从而

$$\begin{equation}
\lambda _n = (n\pi)^{-2}\quad (n \in \mathbb N_+).
\end{equation}$$

对应的特征向量是 $c\cdot \sin n\pi x$ ($c \neq 0$). 

{% endpf %}

{% endex %}

若选取合适的核函数, 则微分方程问题是线性代数问题.

### 子空间与映射

{% def %}
(像, 原像). 给定映射 $f : X \to Y$, 子集 $X' \subseteq X$ 与 $Y' \subseteq Y$, 定义

1. $f(X') := \{y \in Y \mid \exists x \in X', f(x) = y\}$, 称为 $X'$ 在 $f$ 下的像;
2. $f^{-1}(Y') := \{x \in X \mid f(x) \in Y'\}$, 称为 $Y'$ 在 $f$ 下的原像.

{% enddef %}

以下公式的检验较为无聊, 供查漏补缺用.

{% ex %}
以下的线性空间都是某个给定线性空间的子空间. 视个人偏好完成.

1. 给定一族以指标集 $I$ 标记的 $\{U_i\}_{i \in I}$, 则 $\bigcap_{i \in I} U_i$ 也是子空间.
2. 给定一族以指标集 $I$ 标记的 $\{U_i\}_{i \in I}$, 则 $\bigcup_{I_0 \subseteq I} (\sum _{i \in I_0}U_i)$ 也是子空间. 此处 $I_0$ 取遍 $I$ 的有限子集.
3. 给定子空间 $U_1 \subseteq U_2 \subseteq \cdots$, 则 $\bigcup_{i=1}^{\infty} U_i$ 也是子空间.
4. 给定子空间 $U_1 \subseteq U_2 \subseteq \cdots$, 则 $V \cap (\bigcup_{i=1}^{\infty} U_i) = \bigcup_{i=1}^\infty (V \cap U_i)$.
5. 给定子空间 $U_1 \supseteq U_2 \supseteq \cdots$, 则 $V + (\bigcap_{i=1}^{\infty} U_i) \subseteq \bigcap_{i=1}^\infty (V + U_i)$. 找一个无法取等号的例子. (提示: 可从数列空间入手.)

假定 $f : U \to V$ 是线性映射, $U', U''$ 是 $U$ 的子空间, $V', V''$ 是 $V$ 的子空间. 试证明以下命题.

1. $U' \subseteq f^{-1}(f(U'))$;
2. $f(f^{-1}(V')) \subseteq V'$;
3. $f (f^{-1} (f (U'))) = f(U')$;
4. $f^{-1} (f (f^{-1} (V'))) = f^{-1} (V')$;
5. $f(U' + U'') = f(U') + f(U'')$;
6. $f^{-1}(V' \cap V'') = f^{-1}(V') \cap f^{-1}(V'')$;
7. $f((f^{-1}(V')) \cap U') = V' \cap f(U')$;
8. $f^{-1}((f(U')) + V') = U' + f^{-1}(V')$;
9. $f (f^{-1}(U' \oplus U'')) = f (f^{-1}(U')) \oplus f (f^{-1}(U''))$;
10. $f^{-1} (f (V' \oplus V'')) = f^{-1} (f (V')) \oplus f^{-1} (f (V''))$.

{% endex %}

### 基/维数巩固习题

{% pro %}
假定矩阵 $A, B \in \mathbb Q^{2m \times 2m}$ 满足

$$\begin{equation}
A^2 = B^2 = I_{2m},\quad AB+BA = O_{2m}.
\end{equation}$$

试证明, 存在可逆矩阵 $P$ 使得

$$\begin{equation}
PAP^{-1} = \begin{pmatrix}I_m & O \\ O & -I_m\end{pmatrix},\quad PBP^{-1} = \begin{pmatrix}O&I_m \\ I_m & O \end{pmatrix}.
\end{equation}$$

提示: 说明 $A$ 可对角化, 选用合适的基变换.

{% endpro %}

{% ex %}
假定 $f : V \to V$ 是有限维空间的线性映射, 则

$$\begin{equation}
\dim f(V) = \dim (f(V)\cap f^{-1}(0)) + \dim f(f(V)).
\end{equation}$$

{% pf %}
考虑满射

$$\begin{equation}
\varphi : f(V) \to f(f(V)), \quad x \mapsto f(x).
\end{equation}$$

此时 $\dim \varphi^{-1}(0) + \dim \varphi (V) = \dim f(V)$. 由 $\varphi$ 满, 得 $\varphi (V) = f(f(V))$. 另一方面,

$$\begin{equation}
\varphi^{-1}(0) = f(V) \cap \varphi^{-1}(0) = f(V) \cap f^{-1}(0).
\end{equation}$$

综上所述, 即得所需结论.

{% endpf %}

{% endex %}

{% pro %}
假定 $U \xrightarrow f V \xrightarrow g W$ 是有限维线性空间的线性映射. 证明以下问题.

1. $\dim f(U) + \dim g(V) \leq \dim g(f(U)) + \dim V$.
2. $\dim f^{-1}(0) \leq \dim (g\circ f)^{-1}(0) \leq \dim g^{-1}(0) + \dim f^{-1}(0)$.
3. $\dim (g\circ f)(U) \leq \min (\dim g(V), \dim f(U))$.
4. $g^{-1}(g(f(U))) = f(U) + g^{-1}(0)$.
5. $f(f^{-1}(g^{-1}(0))) = f(U) \cap g^{-1}(0)$.

{% endpro %}
