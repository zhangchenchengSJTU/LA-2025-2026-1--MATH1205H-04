## 投影矩阵

{% pro %}
判断题 (假定所有矩阵是 $\mathbb R^{n \times n}$ 中的).

1. 若 $P$ 是投影矩阵, 则 $I - P$ 也是投影矩阵.
2. 投影矩阵的分量必须要有 $0$ 和 $1$.
3. 若 $P$ 是投影矩阵, 则 $\det(P)$ 必为 $0$ 或 $1$.
4. 若 $P$ 是投影矩阵, 则 $\dim C(P) = \mathrm{tr}(P)$.
5. 若 $P, Q$ 是投影矩阵, 则 $C(P+Q) = C(P) + C(Q)$, 此处 $C(\cdot)$ 表示列空间.
6. 若 $P, Q$ 是投影矩阵, 则 $N(P+Q) = N(P) \cap N(Q)$, 此处 $N(\cdot)$ 表示零空间.
7. 若 $P, Q$ 是投影矩阵, 则 $C(PQ) = C(P) \cap C(Q)$.
8. 若 $P, Q$ 是投影矩阵, 则 $N(PQ) = N(P) + N(Q)$.
9. 若 $P$ 是投影矩阵, 则 $N(I - P) = C(P)$.
10. 若 $P$ 是投影矩阵, 则全空间具有直和分解 $N(P) \oplus C(P)$.

{% sol %}
依次解答如下.

1. 正确. 直接验证.
2. 错误. 比如 $\binom{0.5 \ 0.5}{0.5 \ 0.5}$.
3. 正确. 投影矩阵的特征值只能是 $0$ 或 $1$, 因此行列式只能是 $0$ 或 $1$.
4. 正确. 相似变换不改变迹. 依照实对称矩阵的正交相似对角化, 所有投影矩阵正交相似于某个对角矩阵, 其对角线元素为 $0$ 或 $1$. 因此迹等于 $1$ 的个数, 即列空间的维数.
5. 正确. $P+Q = PP^T+QQ^T=(P\mid Q)(P \mid Q)^T$. 对实矩阵, $C(AA^T)=C(A)$.
6. 正确. $N(P+Q) = N(PP^T+QQ^T) = N\binom {P^T}{Q^T} = N(P) \cap N(Q)$. 也可以对上一问的结论使用正交补.
7. 错误. 反例: $P = \binom{1 \ 0}{0 \ 0}$, $Q = \binom{0.5 \ 0.5}{0.5 \ 0.5}$. 则 $C(PQ) = \{\binom k0\}$, 而 $C(P) \cap C(Q) = \{0\}$.
8. 错误. 反例同上. 也可以对上一问的结论使用正交补.
9. 正确. 由 $(I-P)Px = 0$ 知 $C(P) \subseteq (I - P)$. 另一方面, 若 $(I-P)x = 0$, 则 $x = Px \in C(P)$.
10. 正确. 我们知道 $N(A) \oplus C(A^T)$ 总是直和分解. 结合 $P = P^T$ 即可.

{% endsol %}

{% endpro %}

{% pro %}
请用简单的反例说明以下事情:

1. 以下三个条件两两独立: $P^2 = P$, $P = P^T$, 以及 $P^2 = P^T$. 简单地说, 可以找到三个例子, 分别满足其中一个条件但不满足剩下两个条件.
2. $P$ 与 $Q$ 是投影矩阵, 但 $PQ$ 未必是投影矩阵.
3. $P$ 与 $Q$ 是投影矩阵, 但 $P+Q$ 未必是投影矩阵.
4. $P_1, P_2, P_3$ 是投影矩阵, 满足 $P_1P_2 = P_2P_1$ 与 $P_2P_3 = P_3P_2$, 但未必有 $P_3P_1 = P_1P_3$.

{% sol %}
依次给出反例如下.

1. 一阶矩阵 $A = (2)$ 仅满足 $A=A^T$. 矩阵 $B = \binom{1 \ 1}{0 \ 0}$ 仅满足 $B^2 = B$. 三阶轮换矩阵 $C = \begin{pmatrix}0 & 1 & 0\\ 0 & 0 & 1\\ 1 & 0 & 0\end{pmatrix}$ 仅满足 $C^2 = C^T$.
2. 取 $P = \binom{1 \ 0}{0 \ 0}$, $Q = \binom{0.5 \ 0.5}{0.5 \ 0.5}$.
3. 取 $P = Q \neq O$.
4. 任取 $P_1P_3 \neq P_3P_1$, 以及 $P_2 = O$.

{% endsol %}

{% pro %}
回顾线性方程组 $Ax = b$.

1. 简要说明, 方程组有解当且仅当 $AA^+ b = b$.
2. 假定方程组有解, 则解必然形如 $A^+b + (I - A^+A)y$. 此处 $y$ 是任意向量.
3. 假定方程组有解, 则方程组有唯一解当且仅当 $A^+A = I$.

{% sol %}
依次解答如下.

1. $A^+A$ 是向 $C(A)$ 的投影矩阵. 因此 $A^+A b = b$ 等价于 $b \in C(A)$, 即方程组有解.
2. 设 $x_0$ 是某个解. 则任意解可表示为 $x_0 + N(A)$. 注意到 $N(A) = N(A^+A) = C(I - A^+A)$.
3. 方程组有唯一解当且仅当 $N(A) = \{0\}$, 即 $A^+A$ 是满秩的投影矩阵, 即 $I$.

{% endsol %}

{% endpro %}
